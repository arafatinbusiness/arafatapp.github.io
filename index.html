Book Library Genesis

This Android project follows the MVVM architecture pattern for efficient UI and data interaction, with ViewBinding enabled for streamlined view management. SQLite is used for storing book-related data, such as title, author, and page count, with database operations like adding, reading, updating, and deleting handled by the MyDatabaseHelper class. The app allows users to add, search, and display books, offering seamless navigation and updates via RecyclerView and FloatingActionButton, with Toast messages providing user feedback for each operation.

To do list app

This To-Do app integrates RecyclerView and CardView to efficiently display and organize tasks in a user-friendly interface. The app utilizes SQLite as a local database to securely store and manage task data, ensuring persistence across sessions. This project showcases the use of Android components for dynamic UI and data management, offering a robust solution for task organization
